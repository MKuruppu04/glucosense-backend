About This Project

Non-Invasive Glucose Monitoring

This project implements a real-time, non-invasive blood glucose monitoring system based on research in optical sensing technology. Using infrared (IR) spectroscopy combined with photoplethysmography (PPG), we can estimate blood glucose levels without the need for finger pricks or blood samples.

How It Works

The system uses two primary sensing technologies:

  • MAX30102 PPG Sensor: Measures heart rate, blood oxygen saturation (SpO2), and provides red/infrared light absorption data
  • IR Sensor: Detects infrared light absorption patterns that correlate with glucose concentration in blood
